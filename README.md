# 🚀 Deployment Steps

## 1️⃣ Create Project Files

Create:
- `index.html`
- `style.css`
- `README.md`

---

## 2️⃣ Initialize Git Repository

```bash
git init
git add .
git commit -m "Initial commit"
```

---

## 3️⃣ Create GitHub Repository

Create a new repository on GitHub.

Example repository name:

```text
static-website-with-github-pages
```

---

## 4️⃣ Push Code to GitHub

```bash
git remote add origin https://github.com/harshavardhanbhosale/static-website-with-github-pages.git

git branch -M main

git push -u origin main
```

---

## 5️⃣ Enable GitHub Pages

Go to:

```text
Repository → Settings → Pages
```

Then:
- Select Branch = `main`
- Select Folder = `/root`
- Click Save

GitHub will automatically deploy the website.
